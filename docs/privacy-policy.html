Politique de confidentialité

URS-10T Reader — Lecture colorimétrique de bandelettes

En résumé : URS-10T Reader ne collecte aucune donnée personnelle. Toutes vos données restent sur votre appareil. Aucune information n'est transmise à des tiers.

Données collectées

URS-10T Reader ne collecte aucune donnée personnelle. L'application fonctionne entièrement en local sur votre appareil :

  • Aucun compte utilisateur n'est requis
  • Aucune donnée n'est envoyée à un serveur
  • Aucun identifiant n'est collecté

Stockage local

Les données suivantes sont stockées uniquement sur votre appareil :

  • Les photos de vos bandelettes URS-10T
  • Les résultats de lecture colorimétrique
  • Vos notes et le contexte de chaque test
  • Vos préférences (thème, rappels)

Ces données ne quittent jamais votre appareil sauf si vous utilisez volontairement la fonction d'export (PDF, CSV, JSON).

Caméra

L'application utilise la caméra de votre appareil uniquement pour photographier les bandelettes de test URS-10T. Les photos sont traitées localement et ne sont jamais transmises à un service externe.

Tracking et publicité

  • Aucun outil de tracking ou d'analytics n'est utilisé
  • Aucune publicité n'est affichée
  • Aucun cookie n'est utilisé
  • Aucune donnée n'est partagée avec des tiers

Export de données

Vous pouvez exporter vos données en PDF, CSV ou JSON. Ces exports sont initialisés uniquement par vous et restent sous votre contrôle. Aucune sauvegarde automatique vers le cloud n'est effectuée.

Suppression des données

Vous pouvez supprimer vos tests individuellement depuis l'historique. La désinstallation de l'application supprime toutes les données stockées.

Modifications

Cette politique de confidentialité peut être mise à jour. Toute modification sera publiée sur cette page.
